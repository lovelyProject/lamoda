## Lamoda
# Онлайн магазин с базовым функционалом корзины, подсчета суммы, фильтрами, выводом товара с апи  https://fakestoreapi.com/

# Запуск проекта локально

### Установка node 18.16.0
- nvm install 18.16.0
- nvm use 18.16.0

### Установка зависимостей через yarn
- yarn

### Запуск приложения 
- yarn dev
